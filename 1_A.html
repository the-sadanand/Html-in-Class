Understanding Lists in HTML

Lists are used to group related items in HTML. There are different types of lists: ordered, unordered, nested, and definition lists.

Example 1: Ordered List

An ordered list is used when the sequence of items matters:

  1. First item
  2. Second item
  3. Third item

Example 2: Unordered List

An unordered list is used when the sequence of items does not matter:

  • Item A
  • Item B
  • Item C

Example 3: Nested Lists

Lists can be nested inside one another:

  1. Main Item 1
    • Sub-item 1.1
    • Sub-item 1.2
  2. Main Item 2

Example 4: Unordered Lists in an Unordered List

Multiple unordered lists can be nested within each other:

  • Category 1
    • Sub-category 1.1
    • Sub-category 1.2
  • Category 2
    • Sub-category 2.1
    • Sub-category 2.2

Example 5: Definition List

A definition list is used to define terms:

HTML
HyperText Markup Language, used to structure web content.
CSS
Cascading Style Sheets, used to style web content.
JavaScript
A programming language used to add interactivity to web pages.
